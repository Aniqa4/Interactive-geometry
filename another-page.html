What is DOM and what is the purpose of DOM?

DOM stands for Document Object Model. This is a method to connect a programming language to a web document to change it or modify it.

How will you select HTML elements using DOM? Name the DOM methods.

There are five methods to select HTML elements. They are getElementById, getElementsByClassName, getElementsByTagName, querySelector and querySelectorAll

What is event bubble?

When an event occurs on an element, such as a click, the event is first triggered on that element, and then it bubbles up to its parent elements until it reaches the top of the DOM tree. During this phase, any event listeners on each element can be triggered and executed.

What is a callback function and why will you use it?

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete actions.
